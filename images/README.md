# Vehicle Images

Add vehicle images to this directory:

- `volvo.png` - 2021 Volvo XC40 Recharge (blue)
- `equinox.png` - 2024 Chevrolet Equinox EV (gray)

Recommended specs:
- Format: PNG or JPG
- Size: ~400x250px (16:10 aspect ratio)
- Orientation: 3/4 front view
- Background: Light or transparent
